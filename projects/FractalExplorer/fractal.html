Theory

The Mandelbrot set is the set of complex numbers c for which the function fc(zn) = z(n-1)2 + c does not diverge when z0 = 0.

Named after the mathematician Benoit Mandelbrot, its representation is created by sampling the complex numbers and testing for each one of them (c) if the above function, when iterated, makes c diverge to infinity or not.
In practise, we'll consider c to converge if it reaches the maximum iterations.
Similarly, we'll consider c to diverge if its modulus reaches the maximum value.

Treating the real and imaginary parts of c as coordinates on the complex plane, we can then color each pixel according to the number of iterations it took it to diverge.
If it didnt', we just make it black.

The Mandelbrot set is related to Julia sets in such a way that every point in the complex plane corresponds to a whole Julia set.

If the point doesn't belong to the Mandelbrot set then its corresponding Julia set will consist of nothing but isolated points[ref],
otherwise it will be a single, connected figure.

Controls

Use the sliders to control the simulation's parameters :

  • Range : Min and Max values of the axis
  • Step : Difference between two discrete points on the axis
  • Translation : Offset on the axis
  • Iterations : Max iterations
  • Modulus : Max modulus

Click Go to launch the simulation.
Be patient and give it a couple of seconds to compute !

Once it's done, click anywhere in the canvas to display the
Julia set corresponding to that complex number.
