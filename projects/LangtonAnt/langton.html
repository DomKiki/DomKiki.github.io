Theory

The Langton's Ant is a cellular automaton invented by Christopher Langton. It is said to be one of the simplest systems to illustrate emergent behaviour.
The rules are simple indeed : the world is made of colored squares ; the Ant turns right or left according to the color of the square it's on, and changes said color before moving forward.

A set of rules can then be expressed as a scheme of Right and Left, for example LRRL.
The original Langton's Ant (shown above) has the simplest scheme of all : RL. It is nonetheless

Fun fact : although it's never been proven, all finite set of rules will eventually result in what is referred to as a highway, meaning the Ant will build the same recurrent structure indefinitely.

The scheme is here visually represented as a cycle with color nodes for easier understanding of the Ant's behaviour.
It has also been extended to 7 possible behaviours : 90° CW (Right), No change, 90° CCW (Left), UP, RIGHT, DOWN and LEFT.
The first 3 are relative, the next 4 are absolute, meaning they don't depend on the current Ant's direction.

Controls

The left-hand buttons control the simulation progress :

  • Clear and Pause/Resume are self-explanatory
  • Step executes a single move of the Ant
  • Frame executes a whole frame of the Ant

The SpF (Steps per Frame) slider controls how many steps the Ant should perform in a single frame.

The middle cycle allows for changes in the colors and behaviours :

  • Add a new node at the end of the cycle
  • Edit a node's color and the Ant's behaviour
  • Delete a node

The right-hand buttons allow to Add or Remove Ants (randomly initialized).
